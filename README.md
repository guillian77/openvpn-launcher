# How it work ?
- Start VPN connection: `./openvpn.sh start`
- Start VPN connection in interactive mode: `./openvpn.sh start -it`
- Stop VPN connection: `./openvpn.sh stop`
- Get VPN process status: `./openvpn.sh status`
